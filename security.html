Security

How we keep your files safe and secure

100% Local Processing

Friday V2 operates entirely on your Mac. Your files are never uploaded to any server, ensuring complete privacy and security of your data.

Local Only

All file processing happens on your device. No cloud, no servers, no data transmission.

Sandboxed

Runs in a secure macOS sandbox with limited system access.

Undo History

Every action can be undone. Your original file structure is always recoverable.

Transparent

See exactly what Friday V2 will do before confirming any action.

Our Security Practices

Application Security

  • Code Signing: The app is signed with a valid Apple Developer certificate
  • Notarization: Verified by Apple's notarization service
  • Sandboxing: Runs in macOS sandbox with minimal permissions
  • No Root Access: Never requests administrator privileges

Data Security

  • Zero Network: No internet connection required for core functionality
  • No Telemetry: No tracking or analytics without explicit consent
  • Local Storage: All preferences stored locally in encrypted format
  • No File Upload: Files are never transmitted anywhere

Permissions Requested

Friday V2 only requests the permissions it needs:

  • File Access: To read and organize files in selected folders
  • Finder Extension: Optional, for right-click integration

We never request:

  • Camera or microphone access
  • Location services
  • Contact or calendar access
  • Full disk access (only selected folders)

Safe File Handling

Friday V2 includes multiple safety features:

  • Preview Mode: See all proposed changes before applying them
  • Undo Stack: Complete history of all actions for easy reversal
  • Protected Folders: System folders are automatically excluded
  • Conflict Resolution: Smart handling of duplicate file names
  • Dry Run: Test organization rules without moving files

Third-Party Dependencies

We carefully vet all dependencies:

  • Minimal external dependencies
  • Regular security audits
  • No analytics or tracking SDKs
  • Open-source components where possible
