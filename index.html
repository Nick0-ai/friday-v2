Friday is now 20n.

Friday started as an experiment — can a brain control a computer the way a human does, without APIs, without cloud, just vision and reasoning? It worked. So we took it further.

Friday was Phase 1. A local AI agent that controlled any Mac interface autonomously. 100% on-device, zero cloud. The goal wasn't file organization — it was proving that a brain could understand and act on a digital environment in real time.

That proof unlocked a bigger question: if the same architecture can control a screen, can it control a physical body?

The answer is 20n. Same thesis, same principles, moved from pixels to robots.
