Getting Started

Welcome to Friday V2! Follow these steps to install and set up the app.

1

Download Friday V2

Download the latest version from our website. The app is a standard .dmg file for macOS.

2

Install the App

Open the .dmg file and drag Friday V2 to your Applications folder.

3

Launch Friday V2

Open Friday V2 from your Applications folder. On first launch, you may need to right-click and select "Open" to bypass Gatekeeper.

4

Grant Permissions

Friday V2 will ask for permission to access your files. Select the folders you want to organize.

Tip

Start with a single folder like your Downloads or Desktop to get familiar with how Friday V2 works before organizing larger directories.

Your First Organization

Let's organize your first folder with Friday V2.

Step 1: Select a Folder

Click the "Add Folder" button or drag a folder into the Friday V2 window. We recommend starting with your Downloads folder.

Step 2: Analyze

Friday V2 will scan the folder and analyze each file. You'll see a preview of:

  • Number of files found
  • Proposed categories
  • Suggested folder structure

Step 3: Review

Before any files are moved, you can review the proposed organization. Click on any category to see which files will be moved there.

Step 4: Organize

Click "Organize" to apply the changes. Friday V2 will move files into their new folders.

Step 5: Done!

Your files are now organized. You can undo any changes at any time from the History panel.

Custom Rules

Create custom rules to organize files exactly how you want.

Creating a Rule

  1. Go to Settings → Rules
  2. Click "New Rule"
  3. Set your conditions (file type, name pattern, date, size)
  4. Choose the destination folder
  5. Save the rule

Example Rules

  • Screenshots to Screenshots folder: Files matching Screenshot*.png → Screenshots/
  • Old downloads cleanup: Files older than 30 days → Archive/
  • Large videos: Videos larger than 1GB → Videos/Large/

Undo & History

Friday V2 keeps a complete history of all organization actions.

Undo Last Action

Press Cmd + Z or click the Undo button to reverse the last organization.

View History

Open the History panel to see all past actions. You can:

  • View details of each organization session
  • Undo any specific action
  • Restore files to their original locations

Keyboard Shortcuts

  • Cmd + O - Add folder
  • Cmd + Enter - Start organization
  • Cmd + Z - Undo last action
  • Cmd + Shift + Z - Redo
  • Cmd + , - Open settings
  • Cmd + H - Show history
  • Space - Preview selected file

Frequently Asked Questions

Is my data safe?

Yes! Friday V2 operates 100% locally on your Mac. No files are ever uploaded to any server. See our Security page for details.

Can I undo changes?

Absolutely. Every action is recorded in the history and can be undone at any time.

What file types are supported?

Friday V2 can organize any file type. It has enhanced intelligence for documents, images, videos, audio, code, and archives.

Does it work with cloud folders?

Yes, Friday V2 works with iCloud Drive, Dropbox, Google Drive, and other cloud storage folders that are synced to your Mac.

How do I exclude files?

You can add files or patterns to the exclusion list in Settings. You can also hold Option while organizing to manually exclude files.
