v1.0.0 Latest December 3, 2024

The first public release of Friday V2! We're excited to bring AI-powered file organization to your Mac.

New Features

  • AI-powered file classification using on-device machine learning
  • Smart folder creation based on file content analysis
  • Duplicate file detection and management
  • Custom organization rules with pattern matching
  • Full undo history for all organization actions
  • Preview mode to see changes before applying
  • Scheduled automatic organization
  • Finder extension for right-click organization
  • Dark mode support
  • Keyboard shortcuts for power users

System Requirements

  • macOS 12 Monterey or later
  • Apple Silicon or Intel Mac
  • 500 MB available storage
