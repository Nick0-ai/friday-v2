v2.0.0 Latest December 3, 2025
Phase 1: Smart File Organization

The first public release of Friday V2! This is Phase 1 of our vision: AI-powered file organization for your Mac. This is just the beginning.

New Features

  • AI-powered file classification using on-device machine learning
  • Smart folder creation based on file content analysis
  • Automatic file organization with intelligent categorization
  • Duplicate file detection and management
  • Full undo history for all organization actions
  • Preview mode to see changes before applying
  • Custom organization rules
  • Dark mode support

System Requirements

  • macOS 12 Monterey or later
  • Apple Silicon or Intel Mac
  • 500 MB available storage
